# Pomodoro 🍅

📣Boost your working efficiency by using Pomodoro method. 

🎉One of the best tools out there to save u from procastination!🧪

The program runs for 4 iteration before it stops!

1st - 3rd cycle 🥇🥈🥉
Work mode: 25 Mins
Rest mode: 5 Mins

4th cycle
Work mode: 25 Mins
Rest mode: 25 Mins
